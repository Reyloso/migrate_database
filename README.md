## Description

Repositorio para ejemplo de migracion de dataset de una base de datos a otra por medio a una api en nest.js

## Installation

```bash
$ yarn install


$ docker-compose up

```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
